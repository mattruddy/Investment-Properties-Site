# Next.js/tRPC/Tailwind CSS/Next Auth template

## What's included

- Next.js
- tRPC (v10)
- Tailwind CSS
- Next Auth

## Development

```bash
npm run dev
# or
yarn dev
```

## Production

```bash
npm run build && npm run start
# or
yarn build && yarn start
```
